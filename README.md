Getting Started

Download the latest release from the Releases Page.

Extract the downloaded ZIP file to a folder on your desktop (e.g., C:\SpringfieldTuneUp).

Right-click on SpringfieldTuneUp.exe and select "Run as administrator."

(Optional but Recommended) Select the "Create Restore Point" option in the utility.

Click the "Optimize System for Fortnite" button.

Restart your PC (highly recommended for all changes to take effect).
